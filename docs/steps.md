*Steps to producing quality apps:*

1) Several rounds of research to iteratively refine the initial concept/idea.

2) Create a comprehensive 'Project Requirements Document' (PRD).

3) Create a comprehensive 'Project Architecture Document' (PAD) with Mermaid diagrams and database schema.

4) Create a comprehensive phased master execution plan.

you can use this prompt for #4:
https://raw.githubusercontent.com/nordeim/o-scent/refs/heads/main/docs/prompt-plan.md

_Adding more steps essentially forces the AI to think more deeply and reduces "AI slop", which is a pitfall of "vibe coding"._
